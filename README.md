## pre-requisites

  - NodeJS
  - Angular CLI  
  - mongodb

## Install 

  - Download/clone the repo
  - Change the mongodb ports or database in back-end/config/database.js
  - go to back-end folder
      ````
      cd back-end
      npm install
      npm start
      ````
  - go to front-end folder
      ````
      cd front-end
      npm install
      ng serve
      ````
  - go to http://localhost:4200 to access the digital library
